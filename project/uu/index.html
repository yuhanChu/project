学校介绍

美国斯特福大学（Stratford University）的学士及研究生硕士学位获得了美国联邦教育部、美国高等教育委员会指定的专业权威认证机构美国独立院校教育认证委员会（ACICS）的认证，也是中华人民共和国驻美大使馆及中华人民共和国教育部认证的国外院校。

信息系统

Information System

获取学位要求：修满54学分（12门课程）

该课程为那些期望成为IT行业的领军人物而打造。课程建立在实时沟通和解决问题的基础上，使得学生能够更加深入的了解信息技术的理论和软、硬件组件的基本概念。通过学习移动和网络技术，软件和移动应用的发展，数据库管理，信息保障和信息系统安全管理，云计算和虚拟化，系统分析等课程，最终参与相关项目并完成课程计划。

软件工程

Software Engineering

获取学位要求：修满54学分（12门课程）

该硕士研究生项目能够帮助学生在软件开发领域成为出类拔萃的精英。课程着重于如应用软件系统开发，飞机，航天器和医疗系统等世界上极为复杂和严格的应用领域。该课程的授课老师均与相关行业的各大企业，政府机构以及国际公认的研究机构有密切的合作研究项目。
